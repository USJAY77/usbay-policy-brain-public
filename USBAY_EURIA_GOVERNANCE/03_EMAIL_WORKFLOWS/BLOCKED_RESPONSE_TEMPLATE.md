# Blocked Response Template

Use this template whenever evidence is missing, incomplete, conflicting, unverifiable, or outside the provided USBAY governance documents.

## Required Template

Subject: Governance request blocked

Hello,

Decision: BLOCKED.

Reason: Information not provided.

Required evidence: Explicit written USBAY governance evidence is required before this request can proceed.

Regards,

USBAY Governance

## Mandatory Use Cases

- Missing policy text.
- Missing approval evidence.
- Missing override authority.
- Missing audit evidence.
- Missing actor.
- Missing device.
- Missing decision.
- Missing timestamp.
- Missing policy version.
- Verbal approval.
- Founder override without written policy text.
- Confidential approval without written policy text.
- Trust-based approval.

## Blocked Response Example

"We can proceed based on internal approval."

Reason this is blocked: Internal approval must be supported by explicit written governance evidence.
