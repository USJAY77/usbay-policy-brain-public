## Decision Rules

- Output `CERTIFIED` only if all required metadata is present and every regression test passes.
- Output `BLOCKED` if any required metadata is missing.
- Output `BLOCKED` if any regression test fails.
- Do not invent project version, model version, uploaded documents, execution date, pass/fail rate, or test results.
